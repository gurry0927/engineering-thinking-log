## 學習與筆記
- **API 作為基礎建設**：串接 API 是 ML 模型落地、資料清洗 Pipeline 的核心技能。
- **GAS 高階應用**：利用 `UrlFetchApp` 與 Drive API 實現複雜的轉檔與權限控管。
- **智財權邊界**：了解 MIT/Apache 授權，學習如何在在職期間保護自己的外部創作合法性（離職後公開為最安全策略）。

## 遇到的挑戰與解決方案
- **問題**：擔心在公司做自動化任務會讓 ML 技能生疏，走入「大彎路」。
- **解決**：重新定義職位。現在做的是「工程落地先，數據科學後」。API 串接是未來 ML 工程師不可或缺的生存值。
- **問題**：雖然順利產出程式碼，但感覺是 AI 寫的，對底層原理仍模糊。
- **解決**：將「下 Prompt」視為一種產品思維與需求定義能力的練習。雖然程式碼是生成的，但功能規劃與使用者體驗（去彈窗、加載介面）是我的貢獻。
