## 📝 今日事件：技術突破與系統穩定化
- **P-04 準確率飆升**：發現透過在 GCP 資料夾存放「範例圖 + JSON 填寫規則」作為 API Context，能讓 AI 直接模仿正確模式。這不是 RAG，而是 **多模態 In-Context Learning (Few-shot)**。

## 💡 關鍵決策與邏輯
1. **多模態 Few-shot 應用**：不單靠 Prompt 文字描述規則，而是直接提供「正確示範」。AI 在「模仿具體案例」上的表現遠優於「理解抽象規則」。
2. **條件過擬合 (Overfitting) 防護**：
   - 指令中明確區分「參考結構」與「重新推論數值」。
   - 引入「待確認」 fallback 機制，由 AI 主動標記信心不足的欄位。
3. **系統高可用性**：透過 API Key 備援，提升系統作為「產品」的穩定性級別。

## 🚩 遇到挑戰
- **抄答案問題**：AI 有時會直接照抄範例中的數值。透過 Prompt 隔離與資料解耦來減輕。
- **通訊延遲**：多階段呼叫 API 會增加成本與等待時間，但考量「門窗表」的高精度需求，此取捨是值得的。
