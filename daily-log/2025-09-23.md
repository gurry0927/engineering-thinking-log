## 學習與筆記
- **無限制的後端**：LINE Bot 只是接收與發送訊息的管道，後端可以自由整合任何 AI 框架（如 Semantic Kernel）。
- **關鍵挑戰**：
    - **回應超時**：LINE 要求 3 秒內回應，複雜的 AI Pipeline 可能需要非同步處理或改用 Push API。
    - **運算成本**：需要額外的 Model API 金鑰與伺服器資源。

## 遇到的挑戰與解決方案
- **問題**：好奇是否能「想加什麼就加什麼」。
- **解決**：技術上可行，但需考慮 LINE API 的 3 秒回應限制與資源配額。
