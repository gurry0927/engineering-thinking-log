# 30 天補全面試知識：實戰派轉職計畫表

## 📅 目標：週末進修，一年後轉型架構師/資深工程師

| 週次 | 主題 | 具體內容 |
| :--- | :--- | :--- |
| **Week 1** | **Python 肌肉記憶** | List Comprehension, Dict, Set, Function args, `try/except` 異常處理, 常用內置函數. |
| **Week 2** | **資料與通訊基礎** | **SQL JOIN (必考)**, CRUD, HTTP 狀態碼 (200, 404, 500), API 設計規範 (RESTful). |
| **Week 3** | **系統設計與架構** | 模組化設計, Failover 機制, 負載均衡 (Load Balance), 異步處理 (Async) 概念. |
| **Week 4** | **AI/ML 核心基礎** | 深度學習基本概念, CV 演算法 (YOLO) 邏輯, 準確率 vs 信心值 fallback 策略. |

## 🛠️ 學習方式 (給週末進修者)
- **週六 (理論日)**：專注於術語定義、看別人的面試題庫、理解「名詞」背後的意義。
- **週日 (實作日)**：將學到的理論應用到目前的專案或 Side Project 中。例如：把專案的 Sheets 改為 SQL 查詢。
