## 🛠️ 重點專案實績 (Project Highlights)

### 1. AI 工程圖門窗表自動化解析系統 (P-04)
- **挑戰**：解決傳統人工讀取圖紙緩慢且易出錯的痛點。
- **技術**：Python + OpenCV (影像切分) + GPT API (數據提取) + GCP Cloud Run (部署)。
- **成果**：實現全自動化的圖片裁切與文字結構化提取，大幅減少業務端對圖時間。

### 2. 報價與合約全鏈路自動化 (P-01, P-02, P-05)
- **演進**：從初期的 GAS + Sheets 報價單生成，演進到 LineBot 檔案歸檔，最終規劃 **資料庫驅動的 WebApp** 中繼站。
- **關鍵技術**：Line Messaging API、GAS、VBA 格式美化、Database Schema 規劃。
- **成果**：建立了一套「從報價到回簽」的封閉自動化環路，支援一案多版本比較。

### 3. 公司數據整合與業務獎金自動系統 (P-06)
- **核心**：資料清洗與自動更新流程。
- **實績**：整合多個部門亂七八糟的維修與業務手工表單，設計「嚴格搜尋」邏輯，確保獎金計算 100% 準確並自動定時更新。
