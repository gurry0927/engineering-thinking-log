# PRD：BIPV 生命周期成本計算模型 (P-03)

## 1. 專案目標
建立一份可在 Excel 中自動計算 BIPV 生命週期成本 (LCC) 與平準化電力成本 (LCOE) 的工具。支援不同壽命情境（30年/50年）的動態參數模擬。

## 2. 輸入參數 (Inputs)
| 參數名稱 | 預設值 | 備註 |
| :--- | :--- | :--- |
| 折現率 | 3% | 資金成本考量 |
| 建材 CAPEX | 8,000 | 每平方公尺單價 |
| 光電 CAPEX | 5,000 | Cell + 封裝 + 配線 |
| 安裝/BOS | 3,000 | 初始施工費 |
| 光電壽命 | 20 年 | 固定替換週期 |
| 年運維費 (建材/光電) | 0.5% / 1.5% | 依初始 CAPEX 計算 |
| 汰換成本係數 | 70% | 第 20, 40 年更換之成本 |
| 年發電衰減率 | 0.5% | 20年後剩餘效率約 80% |

## 3. 運算邏輯 (Calculation)
1. **現金流分層**：區分建材層與光電層。
2. **時間折現**：所有未來支出 (O&M, 汰換) 均乘以 `1/(1+r)^n`。
3. **退場與殘值**：
    - 殘值按「直線法」比例計算。
    - 若 20 年光電板用於 30 年模型，則期末剩餘 10/20 的價值可回收。
4. **指標輸出**：
    - **LCC (NPV)**：總現值。
    - **EAC**：年化均攤。
    - **LCOE**：每度電成本。

## 4. 驗收標準
- [ ] 僅修改 Inputs 表，其餘表格必須自動同步更新。
- [ ] 30 年與 50 年情境必須在同一 Results 表進行並列比較 (Side-by-side)。
- [ ] 包含發電衰減逻辑，確保 LCOE 的準確性。

## 5. 變更日誌 (Change Log)
模型應保留一個隱藏或副表，記錄主要參數的修改歷史，以防決策基準偏移。
