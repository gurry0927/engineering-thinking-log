# 28 歲代謝修復計畫：工程師的肝與血脂保衛戰

## 核心概念
對於 20 多歲的初階工程師，代謝異常通常是「生活形態」而非「器質性病變」。ALT、膽固醇、尿酸的輕度升高是身體在切換至「高壓職場模式」時的代謝混亂，具備極高的可逆性。

## 詳細筆記

### 1. 肝功能修復 (ALT 改善)
*   **關鍵操作**：睡眠是肝臟修復的唯一硬指標。
*   **目標**：固定 00:00 前入睡，維持 7 小時的高品質睡眠。

### 2. 血脂黃金三角 (LDL/HDL/Cholesterol)
*   **增加 HDL (好膽固醇)**：攝取 Omega-3（深海魚、堅果、亞麻仁油）與中等強度運動。
*   **減少 LDL (壞膽固醇)**：遠離油炸、反式脂肪、高糖飲料。
*   **口訣**：多纖維、少紅肉、要運動。

### 3. 尿酸沖刷法 (Uric Acid Control)
*   **不只是少吃肉**：水分補充比禁食更重要。
*   **公式**：每日飲水量 = 體重 (kg) x 35c.c. 以上。
*   **禁忌**：含琥珀酸的濃湯、高果糖糖漿（含糖飲料）、酒類。

### 4. 28 歲的心理建設
*   **逆轉期**：代謝指標的週期約為 3 個月。只要現在開始調整，年底的複檢有極高機率變回綠字。
*   **打工仔心態應用**：不要為了公司的專案犧牲不可逆的健康。肝是自己的，Code 是公司的。
