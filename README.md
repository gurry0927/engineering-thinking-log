# Engineering Thinking Log

這是一個記錄我轉職工程師後的成長軌跡、技術學習與心態變化的日誌。

## 目錄結構

- `daily-log/`: 每日開發日誌與打卡。
- `learning/`: 技術知識點整理與筆記。
- `reflections/`: 軟實力、職涯思考及心態感悟。
- `templates/`: 用於快速建立新紀錄的 MD 模板。

## 如何使用

1. 從 `templates/` 複製合適的模板。
2. 在對應的資料夾中建立新的 `.md` 檔案。
3. 紀錄完成後，執行 `git add .`, `git commit -m "feat: add new log"`, `git push`。
